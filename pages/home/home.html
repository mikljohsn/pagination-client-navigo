Handling pagination on the client


This small demo, together with this corresponding tutorial provides you with a ready to use pagination example, that you can use in your own projects.

The tutorial linked to above, explains how to design a matching server side API, that can be used with this client side code.


The example missing several nice to have features like:

  • How to let users sort data by clicking on the table header
  • Adding matching request parameters to the client-URL to allow for linking into specific pages,

    like this, that will bring you to page 3, with 10 elements per page, sorted by author, ascending:

    ?page=3&size=10&sort=author,asc

The tutorial include exercises with hints for how to add these steps
